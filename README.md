# National Cancer Institute (national-cancer-institute)

The National Cancer Institute (NCI) is the federal government's principal agency for cancer research and training, part of the National Institutes of Health. NCI provides data and APIs for cancer genomics, clinical trials, and drug information.

## APIs

### NCI Genomic Data Commons API

The GDC API provides access to genomic and clinical data from the NCI Genomic Data Commons, supporting cancer research and precision medicine. Endpoints include status, projects, cases, files, annotations, data download, manifest generation, BAM slicing, and submission.

**Human URL:** [https://gdc.cancer.gov/developers/gdc-application-programming-interface-api](https://gdc.cancer.gov/developers/gdc-application-programming-interface-api)

**Base URL:** https://api.gdc.cancer.gov/
